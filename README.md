## 📌 About

**Alert Police** is a SA-MP script developed using **MoonLoader + mimgui** that detects nearby police players and triggers a visual alert system on screen.

Designed for customization, flexibility, and performance inside SA-MP environments.

---

## ⚙️ Features

- 👮 Police detection by skin ID
- 📡 Configurable detection radius
- 🖥️ On-screen animated alerts
- 🎛️ mimgui configuration menu
- 🎨 Customizable colors (RGB)
- 📍 Adjustable screen position (X / Y)

---

## 🎮 Commands
/irota → Check nearby police
/rrmenu → Open configuration menu

---

## 🧠 Tech Stack

- Lua 5.1
- MoonLoader
- SA-MP Functions
- mimgui
